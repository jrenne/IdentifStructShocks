1 Before starting

This course covers various econometric topics, including linear regression models, discrete-choice models, and time series analysis. It provides examples or simulations based on R codes.

The R codes use various packages that can be obtained from CRAN. Several pieces of code also involve procedures and data from a companion package (AEC). Some of these procedures —those pertaining to VAR models— as well as the associated presentation (in Section 2.3) have been prepared jointly with Kenza Benhima. This AEC package is available on GitHub. To install it, one need to employ the devtools library:

library(devtools)
install_github("jrenne/AEC")
library(AEC)
